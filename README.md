# Big Lottery Fund Dashboard

Dasboard app for displaying:

- Github status
- App server statuses
- CMS server statuses
- Performance metrics

## Getting Started

Install dependencies

```
npm install
```

Create a `.env` file with the following values

```
APP_TEST_ENDPOINT=
APP_LIVE_ENDPOINT=
CMS_APP_NAME=
GITHUB_ACCESS_TOKEN=
PAGESPEED_API_KEY=
```

Run the app

```
npm start
```
